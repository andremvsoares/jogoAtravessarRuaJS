# jogoAtravessarRuaJS

Esse foi o projeto onde aprendi a fazer um jogo utilizando:

Divisão de códigos para organização;
Diminuição de variáveis - aprendendo a utilizar listas;
Criar laços de repetições através do "for" para percorrer as listas;
Adicionamos uma biblioteca externa para simular a colisão;
Adicionamos também sons de colisão/pontos e trilha sonora;
